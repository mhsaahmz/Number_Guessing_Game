# Number Guessing Game (Python)

A simple **number-guessing game** built using Python.

## Description
This is a console-based game where the computer randomly selects a number between 1 and 1000, and the player has to guess it.  
The game provides hints if the guess is too high or too low and counts the number of attempts.  
You can type "exit" anytime to quit the game.

## Features
- Random number generation between 1 and 1000
- Input validation (only numbers allowed)
- Hints for higher or lower guesses
- Count of total attempts
- Option to exit anytime by typing "exit"

## How to Play
1. Run the game using Python 3.
2. Enter your guesses one by one.
3. Follow the hints to guess the correct number.
4. When you guess correctly, the game will display your total number of attempts.
5. Type "exit" to quit at any time.

## Example

Please guess a number between 1 and 1000.
